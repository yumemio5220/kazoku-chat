# KAZOKUChat（家族チャット）

## 概要

- 家族間でグループチャットできる Web アプリケーション

## 技術スタック

- フロントエンド: Next.js TypeScript, TailwindcssS
- バックエンド: Supabase の DB および API

## よく使うコマンド

- ビルド: `npm run build`
- テスト: `npm test`
- 開発サーバー: `npm run dev`
- リント: `npm run lint`

## Git ワークフロー

- 機能ブランチ: `feature/機能名`
- 修正ブランチ: `fix/バグ名`
- メインブランチ: `main`

## テスト方針

- 新機能にはユニットテスト必須
- テストファイル: `*.test.ts`

## 注意事項

- API キーはコミットしない
- 環境変数は `.env` で管理
