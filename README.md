# グループチャットアプリ

Next.js + TypeScript + Tailwind CSS で作成されたグループチャットを可能とするアプリケーションです。

## 機能

- ✅ 複数人での同時チャット機能
- ✅ Supabase 認証およびユーザ名などのアカウント情報の設定機能
- ✅ 未読、既読管理
- ✅ 画像や動画ファイルの送信およびプレビュー
- ✅ レスポンシブデザイン（PC・スマホ・タブレット対応）

## 技術スタック

- **フレームワーク**: Next.js 16 (App Router)
- **言語**: TypeScript
- **スタイリング**: Tailwind CSS
- **状態管理**: React Hooks
- **データ保存**: Supabase
